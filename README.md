# Doodle Jump 

A simple **Doodle Jump–style game** built with **HTML, CSS, and JavaScript**. Jump endlessly across platforms, avoid falling, and try to beat your high score.



## Features

- Canvas-based gameplay  
- Smooth jumping and gravity mechanics  
- Start and restart popups  
- Score tracking  
- Cartoon-style UI using Google Fonts  
- Fully playable in the browser (no installation required)



## How to Play

- Press **Start** to begin the game  
- Use your **keyboard controls** (left/right arrows or A/D depending on implementation)  
- Jump on platforms to stay alive  
- Falling off the screen ends the game  
- Press **Restart** to play again



## Built With

- **HTML** – structure  
- **CSS** – styling and animations  
- **JavaScript** – game logic  
- **HTML Canvas** – rendering
